## Arrays

배열을 유용하게 조작할 수 있는 메서드들을 가진 유틸 클래스

크게 아래와 같은 유형을 지원함
- 이진 탐색(binarySearch)
- 정렬(sort, parallelSort)
- 비교(compare)
- 복사(copyOf)
- 할당(fill)
- 다른 값 찾기(mismatch)
- List 변환(toList)
- Stream 변환(stream)
